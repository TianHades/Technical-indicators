# Trade Sentry 规则参考手册

每条规则的判定标准、触发条件、建议操作。

---

## 技术指标说明

系统在日线和周线两个级别上计算以下指标。各指标的取值范围和含义如下。

### 均线 (MA — Moving Average)

| 项目 | 内容 |
|------|------|
| **是什么** | 过去 N 个交易日收盘价的平均值。MA20=20日均线，MA50=50日均线，MA200=200日均线 |
| **怎么读** | 短期均线在长期均线之上 → 多头排列（看涨）；反之 → 空头排列（看跌） |
| **日线参考** | MA20 反映半个月趋势，MA50 反映一个季度趋势，MA200 反映一年趋势 |
| **周线参考** | 周线 MA20 反映约 5 个月趋势，MA50 约 1 年趋势 |
| **价格与均线关系** | 价格 > MA20 → 短期偏强；价格 < MA20 → 短期偏弱；价格在 MA200 附近 → 长期关键位 |

### ADX (Average Directional Index — 平均趋向指数)

| 项目 | 内容 |
|------|------|
| **是什么** | 衡量趋势强度的指标，不判断方向。值越大趋势越强 |
| **怎么读** | <20：震荡市（趋势策略容易亏损）；20-25：灰色地带；>25：趋势市 |
| **与方向的关系** | ADX 高 ≠ 一定涨，也可能在跌。方向看均线排列和价格位置 |
| **系统用法** | ADX>25 + 均线多头 → 上涨趋势；ADX>25 + 均线空头 → 下跌趋势；ADX<20 → 震荡 |

### RSI (Relative Strength Index — 相对强弱指标)

| 项目 | 内容 |
|------|------|
| **是什么** | 衡量近期涨跌力度的摆动指标，范围 0-100。系统使用 14 日 RSI |
| **数值解读** | 80+：极度超买（追涨极高风险）；70-80：超买偏热（不宜追）；40-60：中性（多空平衡）；30-40：偏弱；20-30：超卖偏低；<20：极度超卖（恐慌割肉） |
| **举例** | RSI=57 表示近期涨势略强于跌势但不极端；RSI=42 表示偏弱但非恐慌；RSI=32 表示接近超卖区 |
| **系统用法** | RSI>80+买入 → BLOCK（M01）；RSI<20+卖出 → BLOCK（M02）。LLM 审查中作为判断追涨/恐慌情绪的重要参考 |

### MACD (Moving Average Convergence Divergence)

| 项目 | 内容 |
|------|------|
| **是什么** | 系统输出的 `MACD柱` 是快线（EMA12-EMA26）与信号线（EMA9）的差值。数值因股价不同而异，没有固定"正常范围" |
| **看涨/看跌** | 正值 = 多头动能；负值 = 空头动能。正负切换 = 金叉/死叉 |
| **变化趋势** | 柱值由负转正且逐日变大 → 多头增强；由正转负且逐日变小 → 空头加剧；柱值接近零 → 即将变盘 |
| **日线示例** | +2.5 = 短期多头动能较强；-0.3 = 空头动能微弱接近变盘；+0.001 = 即将死叉 |
| **周线特殊** | 周线值通常比日线大（周波动大）。绝对值 >2 = 力度显著；<0.5 = 动能衰减 |
| **顶背离** | 价格创近 20 日新高，但 MACD 柱峰值 < 前期峰值 → M03 BLOCK |
| **底背离** | 价格创近 20 日新低，但 MACD 柱谷值 > 前期谷值 → M04 BLOCK |
| **LLM 参考** | 周线 MACD 柱为负且绝对值 >2 → 中期空头明确，买入需谨慎；卖出时反而支持离场决策 |

### KDJ (Stochastic — 随机指标)

| 项目 | 内容 |
|------|------|
| **是什么** | 由 K 线、D 线、J 线三条线组成。衡量收盘价在近期高低点区间内的位置，范围 0-100 |
| **怎么读** | K>80 且 D>80：超买区（短期过热）；K<20 且 D<20：超卖区（短期过冷）；K 上穿 D：金叉（看涨）；K 下穿 D：死叉（看跌） |
| **J 线** | J=3K-2D，更敏感。J>100：极度超买；J<0：极度超卖 |
| **日线 KD 值参考** | >80 偏高，<20 偏低；>90 或 <10 属极端 |
| **系统用法** | 周线 J>90 或 K/D>80 → 严重超买，LLM 会提示追高风险；周线底背离/金叉 → 潜在反转信号 |

### 布林带 (Bollinger Bands)

| 项目 | 内容 |
|------|------|
| **是什么** | 以 MA20 为中轨，上下各两个标准差形成上轨和下轨。价格约 95% 时间在带内运行 |
| **怎么读** | 价格触上轨 → 短期偏贵（但可能趋势延续）；价格触下轨 → 短期偏便宜；带宽收窄 → 变盘前兆（缩窄后通常有大波动） |
| **带宽** | 带宽 = (上轨-下轨)/中轨×100%。收窄到极低 → 即将选择方向；扩张 → 波动加大 |
| **系统用法** | 价格触及上轨+带宽>5% → 短期过涨；触及下轨+带宽>5% → 短期过跌。LLM 分析中结合布林带位置综合判断 |

### ATR (Average True Range — 平均真实波幅)

| 项目 | 内容 |
|------|------|
| **是什么** | 衡量价格波动幅度的指标，值越大波动越大 |
| **怎么读** | ATR 绝对值取决于股价。100 元股票 ATR=5 意味着日均波动约 5%（正常）；ATR 分位数 >80% → 当前处于高波动期 |
| **止损参考** | 建议止损幅度 = 2×ATR。如 ATR=5，止损设在入场价下方 10 元 |
| **系统用法** | ATR 分位数用于判定市场波动率水平（正常/偏高/高波动） |

### 量比 (Volume Ratio)

| 项目 | 内容 |
|------|------|
| **是什么** | 当日成交量 ÷ 近 20 日均量 |
| **怎么读** | =100%：与平时持平；>120%：放量（资金活跃）；<70%：缩量（参与度低） |
| **放量的含义** | 上涨放量 = 资金推动（健康）；下跌放量 = 恐慌抛售（需警惕）；震荡放量 = 方向选择 |
| **缩量的含义** | 上涨缩量 = 动能不足（危险）；下跌缩量 = 抛压减弱（可能企稳） |
| **系统用法** | V01：缩量(<70%)+近关键位 → WARN（无量突破）。LLM 审查中作为市场参与度的重要参考 |

### 支撑位与阻力位

| 项目 | 内容 |
|------|------|
| **是什么** | 支撑位：历史上价格跌到这里就反弹的位置（买方密集区）；阻力位：历史上价格涨到这里就被压回来的位置（卖方密集区） |
| **怎么算** | 近 60 日最高点（≥2次触碰确认）= 阻力位；最低点 = 支撑位。MA200 附近叠加强化 |
| **怎么用** | 接近支撑位买入 = 有安全边际；接近阻力位买入 = 容易被压回；突破阻力位且放量 = 有效突破 |
| **系统用法** | S01：距关键位 <2% → 警告或拦截；T01：逆势时是否接近关键位判断拦截级别 |

### K线形态 (Candlestick Patterns)

| 项目 | 内容 |
|------|------|
| **是什么** | 由 K 线的开盘、收盘、最高、最低价组合成的图形，反映多空博弈结果。系统识别 62 种经典形态 |
| **看涨形态** | 锤子线（下影线长，底部反转信号）、启明星（三根K线，底部反转）、吞没形态（阳线吞没前阴线）、刺透形态等 |
| **看跌形态** | 流星线（上影线长，顶部反转信号）、黄昏之星（三根K线，顶部反转）、乌云盖顶（阴线覆盖前阳线）、上吊线等 |
| **中性形态** | 十字星（多空平衡，方向不明）、纺锤线（小实体+长影线，犹豫信号） |
| **系统用法** | 日线和周线分别检测最近 3 根 K 线的形态。LLM 分析中对比日线和周线形态是否一致，不一致则提示矛盾 |

## 趋势环境

### T01 逆势操作

| 项目 | 内容 |
|------|------|
| **含义** | 在下跌趋势中买入，或在上涨趋势中做空——"跟市场对着干" |
| **判定** | ① 趋势方向与操作方向相反 → **BLOCK**；② 反方向但接近关键支撑/阻力位 → 降为 **WARN** |
| **关键位判断** | 距支撑/阻力 <2% 视为"接近" |
| **为什么** | 逆势交易胜率天然低。除非在明确的转折点（支撑/阻力位），否则是赌博 |
| **如何通过** | 等待趋势反转信号（ADX回落+均线转向），或在关键位附近操作 |

### T04 ADX 弱趋势

| 项目 | 内容 |
|------|------|
| **含义** | ADX 过低时趋势信号噪音大，不适合使用趋势策略 |
| **判定** | ADX < 20 → **WARN** |
| **ADX 参考** | <20 震荡市，20-25 灰色地带，>25 趋势市 |
| **为什么** | 低 ADX 环境下均线交叉、突破信号频繁假突破 |
| **如何通过** | 识别提示即可，ADX<20 时降低仓位或使用区间交易策略 |

## 超买超卖

### M01 RSI 超买追涨

| 项目 | 内容 |
|------|------|
| **含义** | 在 RSI 极高时买入——追涨 |
| **判定** | RSI > 80 且方向为买入 → **BLOCK** |
| **RSI 参考** | >70 超买区，<30 超卖区，>80 极度超买 |
| **为什么** | RSI>80 意味着短期涨幅过大，回调概率极高。此时买入是"接飞刀"的反面——"追火箭" |
| **如何通过** | 等 RSI 回落到 70 以下再入场 |

### M02 RSI 超卖杀跌

| 项目 | 内容 |
|------|------|
| **含义** | 在 RSI 极低时卖出——恐慌抛售 |
| **判定** | RSI < 20 且方向为卖出 → **BLOCK** |
| **为什么** | RSI<20 意味着短期跌幅过大，反弹概率极高。此时卖出是卖在地板价 |
| **如何通过** | 等 RSI 反弹到 30 以上再考虑卖出 |

### M03 MACD 顶背离

| 项目 | 内容 |
|------|------|
| **含义** | 价格创新高，但 MACD 柱未同步创新高——上涨动能衰竭 |
| **判定** | 近 20 日价格新高但 MACD 柱峰值下降 → **BLOCK** |
| **为什么** | 这是经典的见顶信号。价格涨但动能跟不上，大概率即将回调 |
| **如何通过** | 确认非背离（MACD 柱同步创新高），或等待背离消化后再入场 |

### M04 MACD 底背离

| 项目 | 内容 |
|------|------|
| **含义** | 价格创新低，但 MACD 柱未同步创新低——下跌动能衰竭 |
| **判定** | 近 20 日价格新低但 MACD 柱谷值上升 → **BLOCK**（卖出操作） |
| **为什么** | 经典见底信号。价格跌但卖压不足，可能即将反弹。此时卖出可能割在地板 |
| **如何通过** | 确认非背离，或等待背离消化后再卖出 |

## 关键位

### S01 阻力位追高 / 支撑位割肉

| 项目 | 内容 |
|------|------|
| **含义** | 在阻力位附近买入（容易被压回来），或在支撑位附近卖出（可能卖在底部） |
| **判定** | 距关键位 <2%：首次/二次触碰 → **WARN**；多次（≥3次）测试未突破 → **BLOCK** |
| **支撑阻力计算** | 近 60 日最高点（≥2次触碰确认）= 阻力位；最低点 = 支撑位。MA200 叠加强化 |
| **为什么** | 关键位是市场的"记忆"。阻力位附近卖压集中，支撑位附近买盘集中 |
| **如何通过** | 等待有效突破（收盘站上关键位+次日回踩确认），或远离关键位（>2%）操作 |

## 成交量

### V01 无量突破

| 项目 | 内容 |
|------|------|
| **含义** | 价格接近关键位但成交量萎缩——突破缺乏资金确认 |
| **判定** | 距关键位 <2% 且当日量 < 20日均量的 70% → **WARN** |
| **为什么** | 缩量突破关键位通常是假突破。真正的突破需要增量资金推动 |
| **如何通过** | 等待放量（量比 >1）后再操作，或确认突破有量配合 |

## 止损

### STOP01 无止损计划

| 项目 | 内容 |
|------|------|
| **含义** | 买入时没有设置止损价 |
| **判定** | **BUY**: stop_loss 为空 → **WARN**。**SELL**: 无条件 PASS（卖出无需止损） |
| **建议** | 止损幅度建议：入场价 - 2×ATR(14)。例如 ATR=5 元，止损设在入场价下方 10 元 |
| **为什么** | 买入没有止损 = 没有风控。亏损可能无限放大。卖出是平仓离场，不适用止损 |

## 交易频率

### F01 过度交易

| 项目 | 内容 |
|------|------|
| **含义** | 当日交易次数超出正常范围 |
| **判定** | 当日**已确认执行**的交易次数 ≥ `max_trades_per_day`（默认 5） → **BLOCK**。仅审查未确认的不会计数 |
| **为什么** | 高频交易通常是情绪驱动的——追涨杀跌、报复性交易。纪律交易者每天交易笔数有限 |
| **如何通过** | 等明天。如果确实需要提高上限，在 config.yaml 中调整 `max_trades_per_day` |

### F02 交易间隔过短

| 项目 | 内容 |
|------|------|
| **含义** | 同一标的两次操作间隔太短——可能是冲动交易 |
| **判定** | 距上次同标的审查 < 用户配置的 `cooldown_minutes`（默认 30 分钟） → **WARN** |
| **注意** | v1 简化版：只看审查提交时间差，不依赖实际执行结果 |
| **为什么** | 同一标的频繁操作往往是情绪化交易——刚止损就反手、追涨后立即加仓 |

## 仓位

### SZ01 单笔仓位超限

| 项目 | 内容 |
|------|------|
| **含义** | 单笔交易仓位超过上限 |
| **判定** | 计划仓位 > `max_position_pct`%（默认 20%） → **BLOCK** |
| **为什么** | 集中持仓 = 集中风险。一笔重仓亏损可能摧毁整个账户 |

### SZ02 总仓位超限

| 项目 | 内容 |
|------|------|
| **含义** | 所有持仓加起来超过上限 |
| **判定** | 当前持仓 + 计划仓位 > `max_total_position_pct`%（默认 80%） → **BLOCK** |
| **注意** | 需要用户主动输入当前持仓比例。跳过时此规则自动 PASS |
| **为什么** | 保留一定现金应对机会和风险 |

### SZ03 仓位异常放大

| 项目 | 内容 |
|------|------|
| **含义** | 本次仓位明显超过自己的历史平均水平 |
| **判定** | 本次仓位 > 近 10 笔均值的 1.5 倍 → **WARN** |
| **为什么** | 仓位突然放大往往是过度自信的信号——特别是刚赚了一笔之后 |
| **注意** | 需要 ≥10 笔历史记录才能激活 |

## 情绪（已移至 LLM 审查）

> **E01/E02 已移除。** 用户自评情绪不可靠——真正情绪化的人不会意识到自己情绪化。
> 情绪和认知偏误的判断现在由 LLM 从交易理由文本中推断：FOMO、恐慌、贪婪、报复性交易、
> 确认偏误、过度自信、锚定效应、损失厌恶等。

## 市场状态分类

| 状态 | 判定条件 |
|------|---------|
| **上涨趋势** | ADX > 25 且均线多头排列 (MA20 > MA50 > MA200) |
| **下跌趋势** | ADX > 25 且均线空头排列 |
| **震荡偏空** | ADX < 20 且日线周线方向一致向下 |
| **震荡偏多** | ADX < 20 且日线周线方向一致向上 |
| **震荡** | ADX < 20 且方向不明确 |
| **灰色地带** | ADX 20-25：参考周线趋势判定，周线明确向下→下跌趋势，向上→上涨趋势，否则→震荡 |
| **正常波动** | ATR 分位数 < 60 |
| **波动偏高** | ATR 分位数 60-80 |
| **高波动** | ATR 分位数 > 80 |

## Verdict 等级

| 等级 | 含义 | 用户操作 |
|------|------|---------|
| **PASS** | 未发现明显问题 | 可以执行 |
| **WARN** | 存在值得关注的风险 | 建议调整后执行，或确认风险后执行 |
| **BLOCK** | 存在严重问题 | 强烈建议修改计划。可强制覆盖（需确认+记录） |

---

## 评分

- **规则评分**（无 LLM 时）: `max(1, 10 - WARN数 × 1 - BLOCK数 × 3)`
- **LLM 评分**: 自然语言综合判断（1-10），不使用固定公式。BUY 和 SELL 分别从不同角度评估
